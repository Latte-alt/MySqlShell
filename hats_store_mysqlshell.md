# Однотабличная база данных по предметной области «Магазин "Головные уборы"» для MySQL Shell

Ниже приведён готовый вариант **однотабличной базы данных** по предметной области **Магазин "Головные уборы"** для **MySQL Shell**.

Сделано с разделением на понятные блоки:
- создание базы данных
- создание таблицы
- добавление данных
- проверка данных

## 1. Создание базы данных

```sql
DROP DATABASE IF EXISTS hats_store_db;
CREATE DATABASE hats_store_db
CHARACTER SET utf8mb4
COLLATE utf8mb4_unicode_ci;

USE hats_store_db;
```

---

## 2. Создание таблицы

```sql
CREATE TABLE hats_products (
    product_id INT AUTO_INCREMENT PRIMARY KEY,
    product_code VARCHAR(20) NOT NULL UNIQUE,
    product_name VARCHAR(100) NOT NULL,
    category VARCHAR(50) NOT NULL,
    brand VARCHAR(50) NOT NULL,
    season_type ENUM('Зима', 'Весна', 'Лето', 'Осень', 'Демисезон') NOT NULL,
    gender_type ENUM('Мужской', 'Женский', 'Унисекс', 'Детский') NOT NULL,
    size_label VARCHAR(20) NOT NULL,
    material VARCHAR(100) NOT NULL,
    color VARCHAR(50) NOT NULL,
    country_of_origin VARCHAR(50) NOT NULL,
    supplier_name VARCHAR(100) NOT NULL,
    purchase_price DECIMAL(10,2) NOT NULL,
    selling_price DECIMAL(10,2) NOT NULL,
    quantity_in_stock INT NOT NULL,
    receipt_date DATE NOT NULL,
    availability_status ENUM('В наличии', 'Заканчивается', 'Нет в наличии') NOT NULL,
    shelf_location VARCHAR(30) NOT NULL,
    notes VARCHAR(255)
);
```

## 4. Проверка данных

```sql
SELECT * FROM hats_products;
```

---

## Что хранится в таблице

Таблица `hats_products` содержит:
- код товара
- название товара
- категорию
- бренд
- сезон
- тип покупателя
- размер
- материал
- цвет
- страну производства
- поставщика
- закупочную цену
- цену продажи
- количество на складе
- дату поступления
- статус наличия
- место на полке
- примечания

---

## Полезные запросы для проверки

### Показать все товары

```sql
SELECT * FROM hats_products;
```

### Показать только товары, которых мало

```sql
SELECT product_code, product_name, quantity_in_stock, availability_status
FROM hats_products
WHERE availability_status = 'Заканчивается';
```

### Показать товары, которых нет в наличии

```sql
SELECT product_code, product_name, brand
FROM hats_products
WHERE availability_status = 'Нет в наличии';
```

### Показать летние головные уборы

```sql
SELECT product_name, category, brand, selling_price
FROM hats_products
WHERE season_type = 'Лето';
```

### Посчитать общую стоимость товара на складе

```sql
SELECT SUM(selling_price * quantity_in_stock) AS total_stock_value
FROM hats_products;
```

### Показать товары конкретного поставщика

```sql
SELECT product_code, product_name, supplier_name, quantity_in_stock
FROM hats_products
WHERE supplier_name = 'Fashion Import';
```
