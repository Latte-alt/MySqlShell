## 1. Создание базы данных

Этот блок создаёт саму базу данных:

```sql
DROP DATABASE IF EXISTS autoservice_db;
CREATE DATABASE autoservice_db

```

---

## 2. Создание таблицы

Этот блок создаёт таблицу `service_orders`:

```sql
CREATE TABLE service_orders (
    order_id INT AUTO_INCREMENT PRIMARY KEY,
    order_number VARCHAR(20) NOT NULL UNIQUE,

    client_full_name VARCHAR(100) NOT NULL,
    client_phone VARCHAR(20) NOT NULL,

    car_brand VARCHAR(50) NOT NULL,
    car_model VARCHAR(50) NOT NULL,
    car_year YEAR NOT NULL,
    license_plate VARCHAR(15) NOT NULL,
    vin VARCHAR(17) NOT NULL UNIQUE,

    service_category VARCHAR(50) NOT NULL,
    service_description VARCHAR(255) NOT NULL,

    master_full_name VARCHAR(100) NOT NULL,

    date_received DATE NOT NULL,
    planned_finish_date DATE NOT NULL,
    actual_finish_date DATE NULL,

    spare_parts_used VARCHAR(255),

    work_cost DECIMAL(10,2) NOT NULL,
    parts_cost DECIMAL(10,2) NOT NULL,
    total_cost DECIMAL(10,2) GENERATED ALWAYS AS (work_cost + parts_cost) STORED,

    payment_status ENUM('Не оплачено', 'Частично оплачено', 'Оплачено') NOT NULL,
    repair_status ENUM('Принят', 'В работе', 'Ожидает запчасти', 'Готов', 'Выдан') NOT NULL,

    warranty_months INT NOT NULL,
    notes VARCHAR(255)
);
```

## 4. Проверка данных

Этот запрос показывает содержимое таблицы после заполнения:

```sql
SELECT * FROM service_orders;
```

---

## Что есть в этой базе

Таблица `service_orders` хранит:

- номер заказа
- ФИО и телефон клиента
- данные автомобиля
- категорию и описание услуги
- мастера
- даты приема и завершения
- использованные запчасти
- стоимость работ и запчастей
- итоговую стоимость
- статус оплаты
- статус ремонта
- гарантию
- примечания

---

## Полезные запросы для проверки

### Показать все заказы

```sql
SELECT * FROM service_orders;
```

### Показать только машины в ремонте

```sql
SELECT order_number, client_full_name, car_brand, car_model, repair_status
FROM service_orders
WHERE repair_status IN ('Принят', 'В работе', 'Ожидает запчасти');
```

### Показать полностью оплаченные заказы

```sql
SELECT order_number, client_full_name, total_cost, payment_status
FROM service_orders
WHERE payment_status = 'Оплачено';
```

### Посчитать общую выручку по выданным заказам

```sql
SELECT SUM(total_cost) AS total_revenue
FROM service_orders
WHERE repair_status = 'Выдан';
```

### Показать заказы конкретного мастера

```sql
SELECT order_number, client_full_name, service_description, repair_status
FROM service_orders
WHERE master_full_name = 'Кузнецов Андрей Олегович';
```
